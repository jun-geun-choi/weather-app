## 프로젝트 설정 방법
1. 레포지토리 클론
```bash
  git clone https://github.com/jun-geun-choi/weather-app.git
  cd weather-app
```
2. 패키지 설치
```
npm install
```
3. 환경 변수 설정
- 프로젝트 루트 위치에 .env.local 파일을 생성하고 아래 내용 추가!
```
OPENWEATHER_API_KEY=56b7ac48c8ec5f4b083290f06f2f0f29
```

## 프로젝트 실행
1. 개발 서버 실행
```
npm run dev
```
2. 브라우저 접속
```
http://localhost:3000
```
